  Este repositório contém os arquivos .html, .css e .js que compõem meu projeto semestral da disciplina de 
Desenvolvimento Web 1.
  O projeto consiste em implementar um site de 3 páginas com estilo criado pelo aluno, contendo uma página
principal (home) e duas páginas que implementam serviços designados pela professora.
  Foi minha primeira experiência tanto concebendo um site, quanto com as linguagens utilizadas, e também 
diversos conceitos de front-end. Enquanto foi divertido e recompensador para mim, não consegui implementar 
todas as funcionalidades que o projeto determinava, minha maior dificuldade foi referenciar os elementos html
dentro do script JS, eu nunca tinha ouvido falar em DOM.
